- you can create debug configuration from different workspaces/projects (python/debug)
- you can create virtual environment:
	check current enabled python version: "py -3 --version"
	create env: "py -3 -m venv .venv"
	activate environment: ".venv\scripts\activate" (in selected env can contain libraries)
